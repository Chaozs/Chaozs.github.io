# React-portfolio

Portfolio link -> https://Chaozs.github.io/Chaozs.github.io/

## Setup

Install dependencies:

```bash
npm install
```

## Development

Start the dev server:

```bash
npm run start
```

## Build

Create a production build:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Deploy

Publish the `dist` folder to GitHub Pages:

```bash
npm run deploy
```
